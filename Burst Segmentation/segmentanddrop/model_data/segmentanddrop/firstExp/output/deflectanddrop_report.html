deflectanddrop Report

Model deflectanddrop

Description
This fourth assignment simulates an average day on a container terminal. It thereby focuses on a small part of the whole. Trucks arrive from outside the terminal in order to deliver a container to or to pick up a container from the container terminal. The former are called export containers while the latter are called import containers. This assignment takes the time span between two truck arrivals into consideration and is a model applying the process-oriented simulation approach. !!! This fourt assignment simulates three container blocks each represented using a desmoj.Stock object !!!
